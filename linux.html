Коротка інформація про ОС GNU/Linux
GNU/Linux — це вільна операційна система, заснована на ядрі Linux, розробленому Лінусом Торвальдсом у 1991 році. Вона складається з різноманітних дистрибутивів, які включають різноманітне програмне забезпечення, бібліотеки та інструменти, розроблені в рамках проекту GNU. Linux відзначається високим рівнем безпеки, стабільності, масштабованості, а також активною підтримкою спільноти розробників. Використовується в різних сферах: від серверів і суперкомп'ютерів до мобільних пристроїв та вбудованих систем.
Перелік найбільш популярних дистрибутивів

Ubuntu

Один із найпоширеніших дистрибутивів, орієнтований на простоту використання. Він підходить для новачків і має широке співтовариство підтримки. Ubuntu базується на Debian і випускається в різних варіантах (для робочих станцій, серверів та хмарних сервісів).

Debian

Стабільний і надійний дистрибутив, відомий своїм консервативним підходом до оновлень і високою якістю пакетів. Debian є основою для багатьох інших дистрибутивів, таких як Ubuntu. Підходить для серверів і просунутих користувачів.

Fedora

Офіційний дистрибутив, підтримуваний Red Hat. Fedora фокусується на нових технологіях і регулярних оновленнях, що робить його популярним серед розробників і для тестування нових функцій.

Arch Linux

Дистрибутив для досвідчених користувачів, який дозволяє створювати систему з нуля, встановлюючи лише необхідні компоненти. Відомий своєю філософією KISS (Keep It Simple, Stupid) і доступом до найновіших програмних пакетів через модель "rolling release".

Linux Mint

Дистрибутив на основі Ubuntu, орієнтований на користувачів, які переходять із Windows. Відзначається зручним інтерфейсом та простотою налаштування, що робить його чудовим вибором для новачків.

CentOS

Дистрибутив, заснований на Red Hat Enterprise Linux (RHEL). Використовується переважно для серверів завдяки своїй стабільності і тривалій підтримці. Був популярним у корпоративному середовищі до оголошення про закінчення його підтримки в 2021 році.

openSUSE

Орієнтований як на робочі станції, так і на сервери. Відомий своєю системою налаштувань YaST і надійністю. Випускається у двох версіях: Leap (стабільні релізи) і Tumbleweed (rolling release).

Manjaro

Дистрибутив, заснований на Arch Linux, але орієнтований на простоту встановлення та використання. Пропонує користувачам доступ до нових технологій Arch, але з більш дружнім підходом до початкових налаштувань.
Приклади команд роботи в Linux CLI (pwd, echo, ls, cd, touch, mkdir, cp, mv, rm, history)

1. pwd (Print Working Directory)

Виводить поточний каталог, у якому ви знаходитеся.
pwd

2. echo

Виводить текст у терміналі. Може використовуватися для створення файлів з текстом.
echo "Hello, Linux!"

3. ls

Виводить список файлів і директорій у поточному каталозі.
ls

4. cd (Change Directory)

Змінює поточний каталог.
cd /home/user/Documents

5. touch

Створює порожній файл або змінює часові мітки існуючого файлу.
touch newfile.txt

6. mkdir (Make Directory)

Створює нову директорію.
mkdir newfolder

7. cp (Copy)

Копіює файл або директорію.
cp file.txt /home/user/backup/

8. mv (Move)

Переміщує або перейменовує файл/директорію.
mv oldname.txt newname.txt

9. rm (Remove)

Видаляє файл або директорію (з опцією -r для рекурсивного видалення директорій).
rm file.txt rm -r directory

10. history

Виводить список попередніх команд, які ви виконували у цьому терміналі.
history
BASH-cкрипт, що імітує авторизацію в системі у текстовому режимі

                             #!/bin/bash

                             # Файл для збереження логінів і паролів
                             USER_FILE="users.txt"


                             register() {
                                 echo "=== Реєстрація ==="
                                 read -p "Введіть логін: " username
                                 # Перевірка, чи вже існує користувач
                                 if grep -q "^$username:" $USER_FILE; then
                                     echo "Користувач з таким логіном вже існує!"
                                     return
                                 fi

                                read -s -p "Введіть пароль: " password
                                echo
                                read -s -p "Підтвердіть пароль: " password2
                                echo

                                 if [ "$password" != "$password2" ]; then
                                     echo "Паролі не співпадають!"
                                     return
                                 fi

                                 # Додавання нового користувача до файлу
                                 echo "$username:$password" >> $USER_FILE
                                 echo "Реєстрація успішна!"
                             }


                             login() {
                                 echo "=== Авторизація ==="
                                 read -p "Введіть логін: " username
                                 read -s -p "Введіть пароль: " password
                                 echo


                                 if grep -q "^$username:$password$" $USER_FILE; then
                                     echo "Авторизація успішна! Вітаємо, $username!"
                                 else
                                     echo "Невірний логін або пароль!"
                                 fi
                             }


                             while true; do
                                 echo "1. Реєстрація"
                                 echo "2. Авторизація"
                                 echo "3. Вихід"
                                 read -p "Виберіть опцію: " choice

                                 case $choice in
                                     1) register ;;
                                     2) login ;;
                                     3) echo "До побачення!"; exit ;;
                                     *) echo "Невірний вибір, спробуйте ще раз." ;;
                                 esac
                             done
